# Xibo Client 빌드 가이드 (Visual Studio 없이)

## 필수 요구사항

.NET Framework 4.8 프로젝트를 빌드하려면 다음이 필요합니다:

1. **MSBuild** (최신 버전)
2. **.NET Framework 4.8 개발자 팩** (타기팅 팩)

## 설치 방법

### 옵션 1: Visual Studio Build Tools 설치 (권장)

Visual Studio IDE 없이 빌드 도구만 설치:

1. https://visualstudio.microsoft.com/downloads/ 접속
2. "Build Tools for Visual Studio 2022" 다운로드
3. 설치 시 다음 워크로드 선택:
   - ✅ **.NET desktop build tools**
   - ✅ **.NET Framework 4.8 targeting pack**

### 옵션 2: .NET Framework 4.8 개발자 팩 직접 설치

1. https://dotnet.microsoft.com/download/dotnet-framework/net48 접속
2. "Developer Pack" 다운로드 및 설치

## 빌드 방법

### 방법 1: PowerShell 스크립트 사용 (권장)

```powershell
cd xibo-dotnetclient
.\build.ps1
```

특정 구성으로 빌드:
```powershell
.\build.ps1 -Configuration Release -Platform "Any CPU"
```

### 방법 2: MSBuild 직접 사용

MSBuild 설치 후:
```powershell
cd xibo-dotnetclient
msbuild XiboClient.sln /t:Restore,Build /p:Configuration=Release /p:Platform="Any CPU"
```

## 빌드 출력

빌드 성공 시 다음 위치에 실행 파일이 생성됩니다:
- Release: `bin\Release\XiboClient.exe`
- Debug: `bin\Debug\XiboClient.exe`

## 문제 해결

### MSBuild를 찾을 수 없는 경우
- Visual Studio Build Tools가 설치되어 있는지 확인
- MSBuild 경로가 환경 변수에 추가되어 있는지 확인

### 참조 어셈블리를 찾을 수 없는 경우
- .NET Framework 4.8 개발자 팩이 설치되어 있는지 확인
- 다음 경로에 참조 어셈블리가 있는지 확인:
  `C:\Program Files (x86)\Reference Assemblies\Microsoft\Framework\.NETFramework\v4.8`

### NuGet 패키지 복원 실패
- 인터넷 연결 확인
- NuGet 패키지 소스 확인

## 참고사항

- 이 프로젝트는 WPF (Windows Presentation Foundation) 애플리케이션입니다
- Windows 환경에서만 빌드 가능합니다
- .NET Framework 4.8이 설치되어 있어야 실행됩니다
